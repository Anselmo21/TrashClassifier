# Garbage Detection Using YoloV8

### Optional
1. Install Anaconda
2. Install your favourite IDE
3. Create Virtual Environment by typing the following command
   * _conda create -n env_name python ==3.10_
4. Activate the virtual environment by typing the command
   * _conda activate env_name_

### Mandatory Steps
5. Install the requirement by typing the following command..
    pip install -r requirements.txt_
6. Place images to be classified in the images folder
7. Run inference script: python image_detector.py
